# 🦸 ComicVerse Hub

ComicVerse Hub is a fully responsive front-end comic store web application built using **HTML, CSS, and JavaScript**.
The project delivers a modern e-commerce-style experience where users can browse comics, filter and sort collections, view detailed comic information, and manage a persistent shopping cart — all without a backend.

---

## 🚀 Features

* 📚 Browse a collection of comics dynamically
* 🔍 Filter and sort comics for easier discovery
* 📖 Detailed comic viewer page
* 🛒 Persistent shopping cart using `localStorage`
* 📱 Fully responsive design for desktop, tablet, and mobile
* ⚡ Fast and lightweight front-end architecture
* 🎨 Modern UI with interactive user experience

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript (Vanilla JS)

### Storage

* Browser LocalStorage

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/ojas-codes/comic-verse-hub.git
```

### 2. Navigate into the project directory

```bash
cd comic-verse-hub
```

### 3. Open the project

You can simply open `index.html` in your browser.

Or use VS Code Live Server for a better development experience.
